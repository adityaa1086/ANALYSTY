# ANALYSTY

This repository contains the Flask application for an AI-driven data analysis tool named AnalystAgent. The application allows users to upload sales and customer data, train machine learning models on the data, evaluate the models' performance, and visualize the results using SHAP values.

## Features

- File upload for data processing.
- Data preprocessing and model training.
- Evaluation of model performance with mean absolute error.
- Visualization of SHAP values for model interpretation.
- Generation of random data for different industries with insights visualization.
